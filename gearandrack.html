Assembly of Mechanism

Below I present the assembled mechanism with both parts added. Overall, this simple project was a learning experience for me. After designing this I realized it would have been easier to start with the creation of the gear rack. This way I could reference the gear teeth pattern to a skecth of a circle, and have the the pattern wrap around the edge of the circle. The method I'm describing would be much more efficient because I would not have to start from scratch when making the individual parts.
