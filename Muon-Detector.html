Detector

The main parts of this detector are the silicon photomultiplier (SiPM) printed circuit board (pictured left) and the main printed circuit board (pictured right). The OLED screen (pictured top center) connects to the main PCB, and displays the count number, uptime, and an line with length proportional to the calculated SiPM peak voltage of the last dectection. The SiPM PCB has a plastic scintillator attached to it (the object wrapped in black electrical tape).When a charged particle passes through the scintillator, it may release energy that is then absorbed and emitted as photons. When these photons hit the surface of the SiPM, a small current is created which is then amplified and shaped by the main PCB. If the signal is strong enough, the Arduino Nano that is attached to the main PCB measures the peak voltage and the time this signal was received. The micro-controller records this data directly to a computer via USB.



Here is a closer image of the main PCB. Amidst the various resistors and capacitors, you may notice the interesting device soldered at U2. This part is a rail to rail op-amp. This piece is used in the peak detector circuit, which helps amplify the pulse created by the SiPM. If this pulse was not amplified, the Arduino Nano would not be able to detect it.
