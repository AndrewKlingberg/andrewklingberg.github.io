About

I wanted to learn the basics of computational fluid dyanmics, and after some searching I found resources from Prof. Barba's Computational Fluid Dynamics class at Boston University. These resources give step by step instructions on how to build up to soliving the Navier-Stokes Equation in Python. Although I rewrote the code as I followed along, all credit is given to Prof. Barbara. This activity is NOT my independent work, but a way to teach myself the basics by re-typing the code, and experimenting with it along the way.
